# InventarioFarmacia
Sistema de Inventario de Medicamentos para Farmacia desarrollado en C# con ASP.NET Core.

## Tecnologías utilizadas
- **Lenguaje**: C#
- **Framework**: ASP.NET Core MVC
- **Herramientas**: Visual Studio, Git/GitHub

## Funcionalidades principales
- Gestión de inventario de medicamentos.
- Visualización de stock en tiempo real.
- Sistema de alertas visuales para stock crítico y fechas de vencimiento.

## Cómo ejecutar el proyecto
1. Clona el repositorio: https://github.com/06munozledy-code/InventarioFarmacia.git
2. Abre la solución `InventarioFarmacia.sln` en Visual Studio.
3. Compila el proyecto.
4. Ejecuta la aplicación presionando el botón "Play" (IIS Express).
